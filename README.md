# 🏦 C++ ATM Bank System

This is a simple **Object-Oriented Programming (OOP)** project in C++ that simulates a basic ATM system.

## 🚀 Features
- PIN authentication system
- Deposit money functionality
- Withdraw money functionality
- Balance inquiry
- Menu-driven interface
- Input validation (no negative transactions)

## 🧠 Concepts Used
- Classes and Objects
- Encapsulation
- Constructor
- Functions
- Conditional statements
- Loops (do-while)

## 💻 How it works
1. User enters PIN
2. If PIN is correct, ATM menu opens
3. User can choose:
   - Deposit money
   - Withdraw money
   - Check balance
   - Exit system

## 🔐 Default Credentials
- PIN: 8796
- Initial Balance: 80000

## 📌 Learning Purpose
This project is created for learning C++ OOP basics and building logic for real-world systems.
